Sistemas

ON GRID, OFF GRID e HÍBRIDO

O sistema ON GRID é conectado diretamente à rede elétrica pública. Quando há produção excedente de energia solar, ela é enviada para a rede. OFF GRID é um sistema independente que utiliza baterias para armazenamento. O sistema HÍBRIDO permite o uso combinado de baterias e injeção de créditos na rede elétrica.

BOMBEAMENTO SOLAR

Ideal para propriedades rurais, o sistema utiliza energia solar para operar bombas de água, tornando o sistema mais sustentável. MPPT Integrado, alta durabilidade, controle de níveis de água, uso de bombas e motores, plug and play, conexão gerador/rede e função PV + AC.

CARREGADOR VEICULAR

O carregador BYD, possui design sofisticado e compacto, uso residencial ou público, com carregamento CA ou CC, gerenciamento de carga e autenticação por RFID ou QR code. Faça um projeto do seu eletroposto e tenha rentabilidade em cada abastecimento.

ZERO GRID PARA MERCADO LIVRE

Toda a energia gerada é utilizada no local e não há envio do excedente para a rede elétrica. Usamos esta tecnologia para consumidores do "Grupo A", que através do nosso projeto HÍBRIDO com Mercado Livre e Sistema ZERO GRID, podemos proporcionar uma grande economia mensal.
